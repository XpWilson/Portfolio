Downfall

Overview

Downfall was a university project in which I was tasked to recreate Light’s Out from Tiger Electronics. It was a one-week endeavor to implement all of the intricacies of this game.

This version of Light’s Out follows the rules of the original. It’s a single-player game in which each tile has two states, an On or Off state. The goal of the game is to set all tiles to their Off state. When a player selects a tile, it and its adjacent tiles get toggled to their opposite state. Each time the game restarts there is a randomized board.

Features

The base version of the game uses an inventory system, checkpoint system, objective system, player and camera controllers. One of the core action provided to the player is the use of the gravnull gun. The gravnnull gun allows the player to grab and launch objects. As level designers we were not allowed to edit these systems, instead we designed levels around these concepts.

In my version of SCRAPS, I used two main level mechanics: the activation of pressure plates and object-snapping. These mechanics are taught early on to serve them as a guide on how to complete the level. There are other systems I created such as destructible walls and cutscenes to deliver a more enjoyable experience.
