TicTacToe

Overview

This was a university project in which I was tasked to recreate the classic Tic Tac Toe. It was a one-week endeavor to implement all of the intricacies of this game.

This version of Tic Tac Toe follows all of the rules of the original. Two players take turns selecting tiles on a 3x3 grid. If a player manages to get a matching pattern in a column, row, or diagonal they are declared the winner. If both players fill up the grid with no discernible pattern then it is considered a draw.
