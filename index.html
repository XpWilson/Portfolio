Creative Process

Before starting any project this is what I do to get my ideas out there.

Sketch it out

This is where it all begins any ideas I have in mind soon get sketched on paper. Once a general theme is made I move on to define the project further.

Prototype

At this point, I take my concepts and apply them onto the screen with code. This includes basic character meshes and blocked out areas as placeholders. Testing out mechanics for their functionality is prioritized. This is also the stage in which if an idea is not enjoyable it will get scrapped. If that does occur it means to start back to the drawing board.

Iteration

Once the prototype has reached a state in which it is playable, we can test to see if it's fun. This is a refinery stage meaning if the idea is panning out to what we hoped for, we can re-implement to make it work.

Polish

It is at this point that most ideas have been tested and implemented. From this point until release, all effort is placed into improving performance and visual quality.
